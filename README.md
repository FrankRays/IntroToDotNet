# Intro to .NET

Put simply, this is an introduction to C#.

The solution IntroToDotNet is the syntax:

* Strings
* Value Types
  * Most integer types 
  * Bools
  * Enum
* Date Times
* Conditions
* Operators
* Arrays 
* Collections
* Loops
* Methods
* Classes
* Namespaces
* Scope
* LINQ
* Error Handling

The solution OurFirstSolution is an example solution that includes an MVC, Web API, and class libraries projects.
